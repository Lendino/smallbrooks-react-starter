# `smallbrooks-react-starter`

## Contents

* Starter app based on `create-react-app`.
* Using `smallbrooks-react-scripts` instead of `react-scripts` to add less support.
* Includes semantic-ui with possibility for theming.

## Install & run

* Clone the repo
* Run `npm install`
* Run `npm start`
